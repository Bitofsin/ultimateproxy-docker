1)
Ubuntu and Debian:
---------------------------
> sudo apt install openssl
---------------------------

Centos and Fedora:
---------------------------
> sudo yum install openssl
---------------------------


2)
----------------------------------
> openssl req -newkey rsa:4096 \
            -x509 \
            -sha256 \
            -days 3650 \
            -nodes \
            -out example.crt \
            -keyout example.key
----------------------------------

OR

----------------------------------
> openssl req -newkey rsa:4096 \
            -x509 \
            -sha256 \
            -days 3650 \
            -nodes \
            -out example.crt \
            -keyout example.key \
            -subj "/C=SI/ST=Ljubljana/L=Ljubljana/O=Security/OU=IT Department/CN=www.example.com"
----------------------------------
The fields, specified in -subj line are listed below:
C= - Country name. The two-letter ISO abbreviation.
ST= - State or Province name.
L= - Locality Name. The name of the city where you are located.
O= - The full name of your organization.
OU= - Organizational Unit.
CN= - The fully qualified domain name.


3)
-----------------------------------------------------------------------------------------
> openssl pkcs12 -export -out example.pfx -inkey example.key -in example.crt
-----------------------------------------------------------------------------------------


4)
If you did all aboves commands in the Ultimate Proxy folder, then your example.pfx file should be here.

Please copy the example.pfx to the mapped container folder

Just make sure to add the following to the proxy config to enable SSL listening:

"ProxyCert": "/App/config/example.pfx"
