You can find various way of generating a .pfx file here: https://medium.com/the-new-control-plane/generating-self-signed-certificates-on-windows-7812a600c2d8

Please generate it without password.

Please copy the example.pfx to the mapped container folder

Once generated and placed next to Ultimate Proxy, make sure to add the following to Proxy config:
Ex: "ProxyCert": "/App/config/CertFileName.pfx"
